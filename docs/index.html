Vehicle Emission API

Fuel economy and CO2 emission data for US vehicles

Endpoint

GET /vehicle/{id}?email=your@email.com

Returns fuel economy and emission data for a vehicle by its fueleconomy.gov ID.

Try it: https://vehicle.akte.de/vehicle/47085?email=user@gmail.com

Authentication

Every request requires a valid email address. Disposable email providers are blocked.

Option 1 — Header (recommended):

curl -H "Email: user@gmail.com" https://vehicle.akte.de/vehicle/47085

Option 2 — Query parameter:

curl "https://vehicle.akte.de/vehicle/47085?email=user@gmail.com"

If both are provided, the header takes precedence.

Response

{
  "make": "Toyota",
  "model": "Camry",
  "year": 2024,
  "city08": 22,
  "highway08": 33,
  "comb08": 26,
  "co2": 338,
  "vclass": "Midsize Cars",
  "fuelType": "Regular Gasoline"
}
FieldTypeDescription
makestringManufacturer
modelstringModel name
yearintModel year
city08intCity fuel economy (MPG or MPGe)
highway08intHighway fuel economy
comb08intCombined fuel economy
co2float | nullCO2 emissions in g/mi (null if unavailable)
vclassstringEPA vehicle class
fuelTypestringPrimary fuel type
For electric vehicles, fuel economy values are in MPGe. Check fuelType for "Electricity". The co2 field is null for EVs and some older vehicles where data is unavailable.

Errors

All errors return JSON with an error field:

{"error": "missing email: provide Email header or ?email= query param"}
StatusCause
400Invalid vehicle ID or email format
401No email provided
403Disposable email address
404Vehicle ID not found
429Rate limit exceeded (1000 req/min)
502Upstream error (fueleconomy.gov unreachable)

Rate Limiting

Global limit of 1000 requests per minute across all clients. The counter resets after each time window.
