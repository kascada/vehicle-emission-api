


### Authentifizierung

Statt einem Token wird die E-Mail-Adresse geprüft und wenn positiv in einen flüchtigen Cache geschrieben.
Der Cache hat pro Eintrag eine Verfallszeit.

```
go run main.go check-email test@gmail.com
```
